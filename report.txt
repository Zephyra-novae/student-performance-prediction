 Student Performance Prediction System

Data Science Project Report



 Introduction

Education is one of the most important factors in personal and societal development. Predicting student academic performance helps institutions identify weak students early and provide necessary support.

This project aims to analyze student performance data and build a Machine Learning model that predicts whether a student will Pass or Fail based on academic and lifestyle factors.

 Project Objective

The main objectives of this project are:

* To analyze student academic records
* To identify factors affecting performance
* To visualize performance patterns
* To build a predictive Machine Learning model
* To evaluate model accuracy



Dataset Description

The dataset used in this project contains student exam performance records.

Key Features:

* **Gender** – Student gender
* **Race/Ethnicity** – Student background
* **Parental Level of Education**
* **Lunch** – Standard / Free lunch
* **Test Preparation Course**
* **Math Score**
* **Reading Score**
* **Writing Score**

A new column **Result** was created:

* Pass → Score ≥ 50
* Fail → Score < 50

 Tools & Technologies Used

* Python
* Pandas → Data Handling
* NumPy → Numerical Operations
* Matplotlib & Seaborn → Visualization
* Scikit-learn → Machine Learning
* Jupyter Notebook / VS Code

 Data Preprocessing

The following preprocessing steps were performed:

* Checked missing values
* Created Result (Pass/Fail) column
* Converted categorical data into numeric using Label Encoding
* Selected relevant features for modeling

This step ensured the dataset was clean and ready for Machine Learning.

 Exploratory Data Analysis (EDA)

Several visualizations were created to understand patterns:

### Key Visuals:

* Pass vs Fail distribution
* Reading vs Writing scatter plot
* Feature correlation heatmap

 Insights:

* Students with higher reading & writing scores performed better in math
* Test preparation course positively impacted results
* Strong correlation found between reading and writing scores

Machine Learning Model

A **Logistic Regression** model was used because:

* It is suitable for classification problems
* Easy to interpret
* Works well on structured datasets

 Model Training Process

Steps performed:

1. Feature selection
2. Train-test split (80% training, 20% testing)
3. Model training using Logistic Regression
4. Prediction on test data

 Model Evaluation

The model was evaluated using:

* Accuracy Score
* Confusion Matrix
* Classification Report

Results:

* The model achieved good accuracy in predicting student performance.
* It correctly classified most Pass and Fail cases.
